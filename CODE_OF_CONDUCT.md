# Code of Conduct

## 我们的承诺

为了建设开放、友好、可协作的社区，我们承诺对所有参与者保持尊重，不因经验水平、性别、身份、地域、宗教或其他个人特征区别对待。

## 我们期望的行为

- 使用建设性语言进行讨论
- 尊重不同观点，聚焦事实与技术问题
- 接受合理反馈，并愿意持续改进
- 以项目整体利益为先，避免无效争论

## 不可接受的行为

- 人身攻击、侮辱、骚扰、威胁
- 恶意引战、刷屏、持续破坏讨论秩序
- 未经许可公开他人隐私信息

## 执行与处理

维护者有权对违反本准则的行为进行处理，包括但不限于：

- 警告
- 删除评论
- 限制或拒绝贡献
- 封禁账号

## 适用范围

本准则适用于项目仓库、Issue、PR、讨论区及其他与项目相关的公共沟通场景。
