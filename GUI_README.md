# GUI 使用说明（完整版）

对应文件：`danmu_gui.py`

## 启动

```powershell
.\run-gui.bat
```

或：

```powershell
python .\danmu_gui.py
```

## 操作流程

1. 填写服务地址（可包含 token）。
2. 点击“解析地址”。
3. 输入关键字并搜索。
4. 选择来源后加载剧集，或直接“加入该来源全集”。
5. 在下载队列确认任务。
6. 设置格式/并发/重试/节流/超时。
7. 点击“开始下载”。

## 主要能力

- 搜索与来源筛选
- 单集/全集加入队列
- 队列去重与批量下载
- 下载日志与进度显示
- 支持手动全屏切换：`F11` 切换、`Esc` 退出全屏

## 限流建议

如果服务端返回 `429 Too Many Requests`：

- 提高“节流(ms)”（例如 `1000` 或 `2000`）
- 降低并发（例如 `2` 或 `3`）
- 保留重试（建议 `>=3`）

## 打包

```powershell
.\build-exe.bat
```

输出：
- `dist\iDanmu\iDanmu.exe`

说明：运行 EXE 不需要 Node.js。
